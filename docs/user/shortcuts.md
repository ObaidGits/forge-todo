# Keyboard shortcuts

The Wave 1 shell provides these discoverable shortcuts:

| Action | Windows/Linux | macOS/iOS simulator host |
|---|---|---|
| Search | `Ctrl+K` | `Command+K` |
| Quick capture | `Ctrl+N` | `Command+N` |
| Shortcut help | `Shift+?` | `Shift+?` |

Toolbar buttons expose the same actions, so shortcuts are never the only path. Focus order is deterministic and keyboard tests run on every target CI runner. Feature-specific shortcuts such as note capture, item toggling, opening, and reordering will be documented when those features exist rather than advertised early.
